Javascript设计模式 - 代理模式

代理模式最基本的形式就是实现对另外的对象（本体）进行访问控制，实现与本体相同的接口，也是是一种优化模式。
注意：代理对象不添加新的功能，具体的功能都在本体实现。

基本结构

主要用途

  • 控制对创建开销昂贵对象的访问

心得体会

代理模式是一种优化模式，同时也增加一个新的辅助类，代码复杂度增加倒是不明显也不难理解。
